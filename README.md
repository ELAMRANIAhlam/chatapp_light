## 🚀 Présentation

**chatapp_light** est une application Flutter moderne permettant d’échanger des messages en temps réel.  
Elle utilise **Firebase Authentication**, **Cloud Firestore**, **Firebase Messaging** et **Flutter Local Notifications** pour offrir une expérience fluide et réactive.

Ce projet a été développé dans le cadre du module **Développement Mobile**.

---

## ✨ Fonctionnalités

### 🔐 Authentification
- Création de compte  
- Connexion / Déconnexion  
- Gestion de session  
- Thème d’authentification personnalisé  

### 💬 Messagerie instantanée
- Envoi et réception de messages en temps réel  
- Groupes de discussion  
- Liste des utilisateurs connectés  
- Page de profil utilisateur  

### 🔔 Notifications
- Notifications push via **Firebase Messaging**  
- Notifications locales via **Flutter Local Notifications**  
- Gestion du handler en background  

### 🎨 Interface moderne
- UI responsive  
- Thème clair/sombre  
- Architecture basée sur **Provider**

| Technologie | Rôle |
